# CFP 财务规划六大模块参考

> 本文档供 Agent 在规划决策时引用，帮助判断用户需求覆盖了哪些模块。

## 一、投资规划

**核心问题：** 现有资产如何配置才能在风险可控的前提下最大化收益？

**关键概念：**
- 有效市场假说 → 主动管理 vs 被动投资的争论
- 马科维茨投资组合理论 → 分散化降低非系统性风险
- 风险收益权衡 → 无风险收益 + 风险溢价
- 资产相关性 → 低相关性资产组合降低波动

**检查要点：**
- [ ] 投资目标：收益目标、流动性需求、投资期限
- [ ] 风险承受能力：客观（年龄/收入/资产）和主观（心理）
- [ ] 资产配置：大类资产比例、地理分散、行业分散
- [ ] 产品选择：选被动指数还是主动基金
- [ ] 再平衡策略：频率、阈值、税务影响
- [ ] 费用意识：管理费、销售费、交易成本

**常见场景算法：**
| 场景 | 使用函数 |
|------|----------|
| 现有资金终值测算 | `calc.fv(pv, rate, years, pmt)` |
| 定期定额投资计划 | `calc.pmt(target, rate, years)` |
| 资产配置偏离检查 | `calc.allocation_drift(current, target)` |
| 仓位规划 | `calc.simple_portfolio(amount, goal_return, timeline, risk)` |

## 二、保险规划

**核心问题：** 人生风险如何通过保险转移？

**关键概念：**
- 风险自留 vs 风险转移
- 保险需求分析：死亡风险、疾病风险、意外风险、长寿风险
- 保额估算方法：收入倍数法（5-10 倍年收入）、需求法（覆盖负债+抚养+教育）
- 保费预算：年收入的 10%-20%

**检查要点：**
- [ ] 寿险：主要收入者的身故风险 → 覆盖房贷+子女教育+配偶生活
- [ ] 重疾险：医疗费 + 收入损失补偿（3-5 倍年收入）
- [ ] 医疗险：住院/门诊报销，百万医疗 or 高端医疗
- [ ] 意外险：意外身故/伤残，低成本高杠杆
- [ ] 养老年金：长寿风险 → 保证终身现金流
- [ ] 已有保单分析：重复保障、保额不足、性价比低的产品

**保险产品选择原则：**
1. 先保障后理财（重疾 + 医疗 + 意外 → 年金/分红）
2. 先大人后小孩（主要收入者优先）
3. 保额优先于保费（买够保额比自己预算重要）

## 三、税务规划

**核心问题：** 合法合规的前提下如何降低税负？

**中国个税体系（2024）：**
- 综合所得：工资薪金、劳务、稿酬、特许权
- 7 级累进税率：3% - 45%
- 基本减除：60,000 元/年
- 7 项专项附加扣除（2023 新标准）

**检查要点：**
- [ ] 是否充分利用了所有专项附加扣除
- [ ] 年终奖是否单独计税更优惠（延续至 2027 年底）
- [ ] 夫妻间的扣除分配优化
- [ ] 灵活用工/自由职业者的经营所得筹划
- [ ] 投资收益的税务（股息红利 20%，基金/股票资本利得暂免）

**注意：** Agent 不做税务筹划建议，只做信息提醒。具体税务问题建议咨询专业税务师。

## 四、退休规划

**核心问题：** 停止工作后如何维持理想的生活水平？

**关键概念：**
- 养老金替代率：退休后收入 / 退休前收入（理想 ≥70%）
- 三支柱：社保基本养老金 + 企业年金/职业年金 + 个人养老金
- 安全提取率（4% 规则）：每年提取本金的 4%，保持本金不枯竭
- 长寿风险：活得太久钱不够的风险

**检查要点：**
- [ ] 退休年龄预期（当前 60/55/50，未来可能延迟）
- [ ] 预期寿命（保守按 85 岁，乐观按 90-95 岁）
- [ ] 退休后月度支出（按当前币值估算，考虑通胀）
- [ ] 社保养老金估算（替代率一般 30%-50%）
- [ ] 个人养老金账户（每年 12,000 限额，享受个税优惠）
- [ ] 退休储蓄缺口分析

**常见场景算法：**
| 场景 | 函数 |
|------|------|
| 退休本金需求 | `calc.retirement_corpus_needed(monthly, retire_age, life_expectancy)` |
| 退休缺口分析 | `calc.retirement_gap(current_age, retire_age, life_expectancy, savings, monthly)` |

## 五、遗产规划（高级）

**核心问题：** 身故后如何按意愿分配财产？少交遗产税？

**中国现状：**
- 目前未开征遗产税，但未来可能
- 主要考虑：法定继承顺序、遗嘱形式、赠与 vs 继承
- 保险作为遗产规划工具有避税和隐私优势

**检查要点：**
- [ ] 是否有遗嘱
- [ ] 继承人是否明确
- [ ] 企业主/大额资产持有者的传承安排
- [ ] 保险受益人的指定和更新

> ⚠️ 遗产规划涉及法律问题，Agent 只做信息提示，不给出法律建议。

## 六、现金流与教育规划

**核心问题：** 短期到中期的大额资金需求如何规划？

**检查要点：**
- [ ] 应急金：3-6 个月生活支出（放货币基金）
- [ ] 购房规划：首付比例、月供承受力、公积金利用
- [ ] 子女教育金：学费测算、教育金保险、基金定投
- [ ] 债务管理：信用卡、消费贷等高息负债优先偿还
- [ ] 现金流匹配：收入结构和支出节奏是否匹配

## 跨模块优先级

Agent 在遇到用户需求时，按以下优先级引导：

```
1. 现金流管理（应急金 + 高息债务，先解决近渴）
   ↓
2. 基础保障（保险，防范黑天鹅）
   ↓
3. 退休规划（长期目标，享受复利）
   ↓
4. 投资规划（资产配置优化）
   ↓
5. 教育规划（如有子女）
   ↓
6. 税务优化 + 遗产规划（高级用户，锦上添花）
```
